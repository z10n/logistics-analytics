# 🚚 Logistics Analytics: Delay Prediction & Route Optimization

> Анализ логистических данных для выявления причин задержек и оптимизации маршрутов.  
> Проект создан на основе реального опыта управления логистическими процессами.

## 📊 Ключевые метрики
- **On-time delivery**: ~92% (варьируется по перевозчикам: 78–96%)
- **Средняя задержка**: 0.4 дня (медиана: 0 дней)
- **Топ-фактор задержки**: расстояние >300 км (+0.8 дня к средней задержке)
- **Точность прогноза задержек**: ROC-AUC 0.74 (Random Forest, balanced classes)

## 🔍 Бизнес-инсайты
1️⃣ **Перевозчики**: Разница в on-time rate между лучшим и худшим — 18 п.п.  
   → Рекомендация: пересмотреть SLA с аутсайдерами или перераспределить объёмы

2️⃣ **Маршруты**: 3 направления (WH_Kazan → Ekaterinburg/Novosibirsk) имеют задержку >1 дня в 40% случаев  
   → Рекомендация: протестировать альтернативных перевозчиков или изменить точку отгрузки

3️⃣ **Сезонность**: Пик заказов в пятницу → рост задержек на +0.4 дня  
   → Рекомендация: увеличить штат курьеров в четверг-пятницу

## 🛠️ Стек
- **Данные**: Синтетические (10k заказов) + логика, близкая к реальной
- **SQL**: JOIN, агрегации, работа с датами, оконные функции
- **Python**: pandas, scikit-learn (имбаланс классов, class_weight), plotly, streamlit
- **Инструменты**: Git, VS Code, Jupyter
